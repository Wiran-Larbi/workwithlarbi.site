## About

GiftDiscovery is a native mobile application designed to solve the perennial problem of finding the perfect gift. Built specifically for the Android ecosystem, the app utilizes a preference-based filtering system to match users with curated gift ideas based on recipient personality, occasion, and budget constraints.

The app prioritizes a fluid, native user experience with high-performance list rendering and real-time data synchronization. By integrating a backend for product metadata and user favorites, GiftDiscovery provides a centralized hub for holiday and event planning, ensuring that meaningful gift-giving is both efficient and stress-free.

## Tools & process

- **Native Development** — Developed using Java and Android Studio, leveraging the Android SDK to ensure deep system integration and smooth UI transitions.
- **UI/UX Design** — Utilized XML layouts with Material Design components to create a clean, modern interface focused on product discoverability.
- **Data Persistence** — Implemented local caching and remote data fetching via Firebase to manage user profiles and saved gift collections.
- **Intent Handling** — Integrated external linking capabilities to allow users to navigate directly to retailer pages for finalized purchases.
